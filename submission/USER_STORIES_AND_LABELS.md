# Create these as GitHub Issues with labels

These need to be created manually in the GitHub repo so the reviewer can see five user stories with labels.

## Issue 1
Title: As a visitor, I want to browse a whimsical landing page so that I immediately understand the GiftWrapped brand.
Label: epic

## Issue 2
Title: As a shopper, I want to view a full gift catalogue before login so that I can explore gift ideas quickly.
Label: theme

## Issue 3
Title: As a shopper, I want to search gifts by category and recipient so that I can narrow down the results.
Label: backlog

## Issue 4
Title: As a user, I want to register and log in so that I can save gifts and view my profile.
Label: theme

## Issue 5
Title: As a maintainer, I want the CI/CD workflow to run on every push so that I can catch regressions early.
Label: technical debt
